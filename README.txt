ColorStack Terminal Adventure (Fully Locked Edition)

Start:
  cd ~/ColorStack_Terminal_Adventure/homes/detective
  cat notes.txt

Flow:
  1) Put the first password into suspects_locked/password.txt and run crime_scene/unlock.sh
  2) Put the second password into evidence_locked/password.txt and run evidence_locked/unlock.sh
  3) Put the final password into finale_locked/password.txt and run finale_locked/unlock.sh

Everything inside *_locked/ is hidden until you unlock it.
Use: ls, ls -a, cd, cat, echo, bash
